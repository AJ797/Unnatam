\begin{abstract}
We identify two structural limitations of hybrid state-space--attention language models: state slots within selective SSM blocks evolve independently and never cross-talk before readout, and the model has no mechanism for learned behavioural control during pretraining. We address both with composable architectural modifications. \emph{Intracellular attention} replaces the linear readout of selective state-space blocks with a content-dependent attention over the $\dstate$ state slots, letting otherwise-independent slots interact before each readout. \emph{Hormone routing} injects a learned token-conditional combination of frozen behavioural direction vectors --- extracted from the partially-trained base model itself via document-style contrast pairs --- into the residual stream after every attention block. Both mechanisms use a zero-initialised output gate so that the unmodified hybrid baseline is recovered exactly at initialisation; the model learns when and how strongly to activate each pathway during training. We pretrain a 125M-parameter Mamba/MQA hybrid on FineWeb-Edu under a fixed 2B-token optimisation budget and report validation perplexity and zero-shot accuracy on HellaSwag, LAMBADA, ARC-Easy, ARC-Challenge, and PIQA across seven variants spanning both contributions, their composition, and their ablations, including a control that replaces the extracted hormone bank with random unit vectors of the same shape. Both components add fewer than 1\% additional parameters to the base model and are released as open-source code.
\end{abstract}
