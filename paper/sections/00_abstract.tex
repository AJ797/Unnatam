\begin{abstract}
We introduce two composable architectural modifications for hybrid state-space and attention language models. \emph{Intracellular attention} replaces the linear readout of selective state-space (SSM) blocks with a content-dependent attention over the $\dstate$ state slots, allowing the otherwise-independent slots to interact before each readout step. \emph{Hormone routing} injects a trained linear combination of frozen behavioural direction vectors --- extracted from the base model itself via document-style contrast pairs --- into the residual stream after every attention block. Both mechanisms use a zero-initialised gate so that the model recovers the unmodified hybrid baseline at initialisation and learns when and how strongly to activate each pathway. We pretrain a 350M-parameter Mamba/MQA hybrid on FineWeb-Edu and find that, holding the optimisation budget fixed, the combined model attains a validation loss of \TBD{X.XX} versus \TBD{Y.YY} for the unmodified baseline, with consistent improvements across HellaSwag, LAMBADA, ARC-Easy, ARC-Challenge, and PIQA. Ablation experiments isolate each contribution and show that randomising the hormone direction vectors removes most of the routing benefit, indicating that the specific extracted directions matter and not merely the additional parameters. Both components are implemented in open-source code and add fewer than 1\% additional parameters to the base model.
\end{abstract}
